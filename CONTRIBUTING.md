# Contributing

Thanks for helping improve Recipe Relay.

## Local setup

1. Clone the repository.
2. Open Chrome and go to chrome://extensions.
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select the repository folder.

No build step is required.

## Development notes

- Keep the extension backend-free.
- Do not add remote scripts.
- Do not add analytics.
- Keep permissions minimal.
- Prefer simple, readable JavaScript over a build system unless there is a clear reason to add one.
- Test recipe extraction against multiple sites and the included demo page.

## Pull requests

For a pull request, include:

- What changed
- Why it changed
- How it was tested
- Any privacy or permission impact

## Code style

Use plain JavaScript, HTML, and CSS. Keep user-facing copy short and clear.
